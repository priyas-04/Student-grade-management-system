# Student Grade Management System

## Description
A Java-based console application to manage student grades,
calculate averages, and display results.

## Features
- Add student details
- Calculate grades
- Display results

## Technology
- Java

## How to Run
1. Import project into IDE (Eclipse / IntelliJ)
2. Configure database using schema.sql
3. Run Main.java
4. Note: Execute schema.sql in MySQL before running the application.

## Learning Outcome
- Hands-on Java OOP concepts
- DAO pattern usage
- Basic database integration
